# Order Status Classification using Machine Learning

## Overview
This project demonstrates a basic Machine Learning classification workflow using Python and Scikit-learn. The model predicts the order status based on order-related information.

## Features
- Load an Excel dataset using Pandas
- Handle missing values
- Remove unnecessary columns
- Encode categorical features
- Split data into training and testing sets
- Train a Decision Tree Classifier
- Predict order status
- Evaluate model accuracy

## Technologies Used
- Python
- Pandas
- Scikit-learn
- OpenPyXL

## Project Workflow
1. Load Dataset
2. Data Cleaning
3. Feature Selection
4. Label Encoding
5. Train-Test Split
6. Model Training
7. Prediction
8. Accuracy Evaluation

## Result
The model was trained using a Decision Tree Classifier and evaluated using accuracy score.
